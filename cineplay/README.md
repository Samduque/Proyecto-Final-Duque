# CinePlay

CinePlay es una aplicación web estilo blog que centraliza la información sobre películas y series disponibles en plataformas de streaming populares. Permite a los usuarios explorar contenido, consultar detalles específicos y descubrir dónde verlo.

## Características

- **Gestión de Películas**: Agregar, ver y listar películas disponibles en distintas plataformas.
- **Gestión de Series**: Agregar, ver y listar series disponibles en distintas plataformas.
- **Streams**: Agregar y ver streams para cada película o serie.
- **Interfaz intuitiva**: Un diseño limpio y fácil de usar para facilitar la navegación.

## Tecnologías utilizadas

- **Django**: Framework web de Python para el desarrollo del backend.
- **HTML/CSS**: Para la creación y estilización de las páginas.
- **Bootstrap**: Framework CSS para el diseño responsivo y componentes predefinidos.
- **JavaScript**: Para algunas interacciones en el frontend.
- **SQLite**: Base de datos predeterminada utilizada para almacenar los datos de películas, series y streams.

## Uso

* En la página principal podrás ver las **películas** y **series** destacadas.
* Accede a los formularios para agregar nuevas películas, series o streams en las vistas respectivas.
* Desde el panel de administración de Django, puedes gestionar todas las entidades de la base de datos: Películas, Series y Streams, Usuarios y Perfiles.
  URL de administración: `http://127.0.0.1:8000/admin/`

## Configuración Adicional

Este proyecto requiere la configuración de algunas variables de entorno para su correcto funcionamiento, como por ejemplo:

- `DJANGO_SECRET_KEY`: Llave secreta para la seguridad de Django.
- `DATABASE_URL`: URL de la base de datos si estás usando una base de datos diferente a SQLite.
